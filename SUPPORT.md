# Support

## Getting help

For usage questions, setup help, and general discussion:

- open a GitHub issue using the templates in `.github/ISSUE_TEMPLATE/`
- include module details (`skills-web`, `skills-ui`, `skills-maven-plugin`, `skills-registry`)
- include environment info (Java, Maven, Node versions) and reproducible steps

## Reporting bugs

Use the **Bug report** issue template and include:

- expected vs actual behavior
- minimal repro steps
- relevant logs (without secrets)

## Reporting security vulnerabilities

Do not file security issues publicly.

Please follow the private disclosure guidance in `SECURITY.md`.

## Support expectations

This project is maintained on a best-effort basis.

- no formal SLA is currently defined
- issue triage and responses depend on maintainer availability
- security reports are prioritized for private triage
