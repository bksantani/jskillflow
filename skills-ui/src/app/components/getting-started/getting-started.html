How to Use jskillflow

Integrate these skills directly into your developer workflow using the custom Maven plugin. Configure the plugin in your consumer project's pom.xml file:

pom.xml 
{{ getGenericUsageSnippet() }}
1

Go to the Skill Catalog page and copy the ID of any skill you want to consume.

2

Paste it under the <skills> block in your project's pom.xml.

3

Run mvn clean compile to automatically fetch and extract the skill into target/skills/.
